# JRE-For-Android
Executeable Java Runtimes For Arm64 Android Devices. Especially For LWJGL Sofwares Like PojavLauncher, MojoLauncher, Zalith Launcher, Amethyst Launcher, Copper Launcher And Their Forks

# Installation Guide
- Download The Required JRE File.
- Tap On Import/Add Runtime.
- Select The tar.xz File.
- Wait For It To Load.
- Tap Set Default Or Use It For A Separate Instance Only.

**NOTICE: The Installation Process Is Mainly According To What PojavLauncher,MojoLauncher, Zalith Launcher Etc Do. The Method Unclear About Other LWJGL Sofwares.**
