# JRE-For-Android
Executeable Java Runtimes For Arm64 Android Devices. Especially For LWJGL Sofwares Like PojavLauncher, MojoLauncher, Zalith Launcher, Amethyst Launcher, Copper Launcher And Their Forks

# Purpose
Zalith Launcher, MojoLauncher, Copper Launcher And Amethyst Launcher Have All The Required Java Runtimes. But However I'm Making This Project For Helping People Who Are:

- Trying To Play On Older Versions Of The Minecraft Java Launchers Without Modern JRE Support For Experimental Reasons.
- Trying To Make Their Own LWJGL Projects For Android And Need Help With JRE Support.
- Trying To Get Mordern JRE Support On Older Or Outdated LWJGL Softwares On Android.

**NOTE: SUPPORT IS NOT GUARANTEED**

# Installation Guide
- Download The Required JRE File.
- Tap On Import/Add Runtime.
- Select The tar.xz File.
- Wait For It To Load.
- Tap Set Default Or Use It For A Separate Instance Only.

**NOTICE: The Installation Process Is Mainly According To What PojavLauncher,MojoLauncher, Zalith Launcher Etc Do. The Method Is Unclear About Other LWJGL Sofwares.**
